A gente não coloca nome de tag no css... apenas quando não setamos classes ou ids... que para podermos referenciar as tags, precisamos ai sim apontar a tag pai para alcançar a tag filho... filho de tag             nav img{}
filho de classe          .layer h3 {}

E se vc usar o developer tools do navegador, ele vai dar a tag antes da classe, mesmo que o CSS funcione sem essa exigencia. E a ferramenta faz isso para ser o mais completa possivel para te fornecer informações
